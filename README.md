## 📄 Descripció del projecte
- Tipus de producte: **ROBA** o **ELECTRONICA**  
- Aplicació de descompte del 10% per roba  
- Aplicació de recàrrec del 15% per electrònica  
- Xec jove de 5€ per menors de 18 anys  
- Evita preus negatius  
- Funcions separades:
  - `calcular_total()` → calcula el total segons condicions  
  - `mostrar_total()` → mostra el resultat formatat  
- Preparat per afegir noves funcionalitats amb branches Git  
- Docstrings i comentaris explicatius “per què”  

---

## 📂 Estructura del codi

| Funció | Descripció |
|--------|------------|
| `calcular_total(preu, tipus, edat)` | Calcula el preu final segons tipus i edat. |
| `mostrar_total(preu, tipus, edat)` | Mostra el total final amb missatge clar. |
